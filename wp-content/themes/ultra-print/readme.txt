For Homepage Setup:
====================

1. Go to Dashboard >> Pages >> Add New, Create a new page and name it as "Home" then select the template "Custom Home Page" and publish it.

2. Go to Dashboard >> Settings >> Reading, Select the option of Static Page, now select the page you created to be the homepage.

For Header Section:
===================

1. Go to Customizer >>Theme Settings >> Header Section, add Button text & its link.

For Slider:
============

1. Go to Dashboard >> Pages >> Add New, Create a new Page and name it, add the content and publish the page. Repeat the step for other slider pages.

2. Go to Customizer >> Theme Settings >> Slider Settings, Click the checkbox of Show/Hide Slider and select the pages of slider that you have created.

For Services Section:
======================

1. Go to Dashboard >> Pages >> Add New, Create a new Page for services title and name it, add the content and publish it.

2. Go to Dashboard >> Posts >> Category, add new category for services name it and publish it.

3. Go to Dashboard >> Posts >> Add New, Create a new Post and name it, set it's featured image (Image size 58px x 58px), add the content, assign the service category and publish the page. Repeat the step for other Services posts.

4. Go to Customizer >> Theme Settings >> Services Section, select the service category and publish it.

For About Section:
===================

1. Go to Dashboard >> Pages >> Add New, Create a new Page and name it, set it feature image (Image size 255px x 300px), add the content and publish the page. 

2. Go to Customizer >> Theme Settings >> About Section, select the page of about that you have created.

For Footer Section:
====================

1. Go to Dashboard >> Appearance >> Widgets, here you can add widgets to the footer widgets areas, which can be seen in footer.

For Copyright Text:
====================

1. Go to Customizer >> Theme Settings >> Footer Text, add footer text here.
